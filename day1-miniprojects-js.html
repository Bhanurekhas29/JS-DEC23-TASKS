JS Day-1 Mini Projects 1 to 10

Note: Open the browser console (F12) to see results. These projects focus on Variables, Operators, and Types.

1. Simple Variable Showcase

Instructions: Declares var, let, and const variables and logs their types. Check the script comments regarding constant reassignment.

2. Basic Arithmetic Calculator

Instructions: Performs addition, subtraction, multiplication, division, and modulo on two numbers.

3. Name and Age Greeter

Instructions: Uses prompt() for name and age, then displays an alert() and logs details.

4. Type Comparison Challenge

Instructions: Compares string "10" vs number 10 using == and ===.

5. Logical Operator Tester

Instructions: Evaluates boolean logic using && (AND), || (OR), and ! (NOT).

6. User Confirmation Game

Instructions: Uses confirm() to ask a question and logs a response based on your choice.

7. Counter with Increment/Decrement

Instructions: Demonstrates ++ and --, and the difference between prefix and postfix increments.

8. String and Number Concatenation

Instructions: Combines strings and numbers using + and demonstrates implicit coercion with subtraction.

9. Basic Type Conversion

Instructions: Explicitly converts values to Number, String, and Boolean.

10. console.log() Debugging

Instructions: Simulates a script execution flow by tracking variable states at multiple points.
