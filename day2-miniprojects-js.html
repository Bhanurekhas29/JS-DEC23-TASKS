JS Day-2 Mini Projects 1 to 10

Note: Open the browser console (Press F12 or Right-Click > Inspect) to view the output before running any project.

1. Age Classifier

Instructions: Enter your age as a number. The console will display "Child", "Teenager", or "Adult".

2. Simple Number Guesser

Instructions: Try to guess the hidden number (Hint: it's 7). You can keep guessing until you get it right.

3. Basic Calculator Function

Instructions: Enter two numbers and an operation symbol (+, -, *, /) when prompted.

4. Countdown Timer

Instructions: No input needed. Watch the console for a countdown from 10 down to "Lift off!".

5. Shopping List Display

Instructions: No input needed. This iterates through an array and logs each shopping item.

6. User Profile Lister

Instructions: No input needed. This uses a for...in loop to display properties of a user object.

7. Skip Even Numbers

Instructions: No input needed. This logs numbers from 1 to 10 but skips all even numbers using continue.

8. Limited Login Attempts

Instructions: Enter the password (Hint: it is secret). You have 3 attempts before the account locks.

9. Vowel Counter Function

Instructions: Enter a word or sentence. The console will tell you how many vowels (a, e, i, o, u) it contains.

10. Ternary Status Checker

Instructions: This uses an arrow function and a ternary operator to log "Active" or "Inactive" status.
